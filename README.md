# Rhea for iOS

Native iOS tribunal client. Submit claims, browse proofs, monitor agents — from your phone.

**[Join the TestFlight beta](https://testflight.apple.com/join/BNya22Jg)**

## Tabs

| Tab | Purpose |
|-----|---------|
| Tribunal | Submit claims, watch 3-5 models debate |
| Radio | Live agent communication feed |
| Governor | Token budgets, cost tracking |
| Tasks | Persistent task queue |
| Atlas | 3D knowledge graph |
| Pulse | System health |
| Aletheia | Immutable proof browser |
| Settings | Server URL, auth |

## Auth

Sign in with Apple, Google, or Microsoft. JWT stored in Keychain.

## Build

Requires Xcode 15+, iOS 17+. RheaKit Swift package provides all views.

Part of [TimeLabs NPO](https://github.com/timelabs-npo) open infrastructure.
